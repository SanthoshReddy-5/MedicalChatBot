About First Aid Bot

What is First Aid Bot?

First Aid Bot is an intelligent medical chatbot designed to help users with first aid advice and quick medical information. Built as a Retrieval-Augmented Generation (RAG) application, it provides fast, accurate, and trustworthy guidance during emergencies.

Tech Stack

This bot integrates powerful technologies to deliver advanced medical support:

  • Uses Google Gemini API for generative responses.
  • Employs Sentence Transformer Embedding Model for semantic search.
  • Stores vectors in Pinecone vector database for efficient retrieval.
  • Knowledge powered by The Gale Encyclopedia of Medicine.
